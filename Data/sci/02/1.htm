Le pH

Le pH est une mesure de l’acidité ou la basicité (ou alcalinité) d'une solution.

Le pH est un nombre sans unité compris entre 0 et 14.

  Solution
0 ≤ pH < 7 acide
pH = 7 neutre
7 < pH ≤ 14 basique

Plus le pH est proche de 0, plus une solution est acide.

Plus le pH est proche de 14, plus une solution est basique (ou alcaline).
