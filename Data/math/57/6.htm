Déterminer les quartiles

  1. Je range la liste dans l'ordre croissant des valeurs
  2. Je calcule l'effectif total N
  3. Le 1er Quartile est la valeur de rang:
    r1 = 25% de N
  4. Le 3ème Quartile est la valeur de rang:
    r3 = 75% de N

Exemple

Liste de valeurs:
55, 56, 58, 59, 62, 64, 69, 78

Effectif total = 8

Le 1er Quartile est la valeur de rang 2 (25% de 8)
C'est la valeur 56

Le 3eme Quartile est la valeur de rang 6 (75% de 8)
C'est la valeur 64
