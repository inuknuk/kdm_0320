Signe d'un produit de plusieurs facteurs

Pour trouver le signe d'un produit de plusieurs facteurs, compte les facteurs ayant un signe négatif !

- si le nombre de facteurs négatifs est pair, le produit est positif.

Exemple :

5 × (-4) × 2 × (-9) = +360 > 0   (+)

- si le nombre de facteurs négatifs est impair, le produit est négatif.

Exemple :

(-3) × (-2) × (-6) = -36 < 0   (-)
