Regrouper

On regroupe ensemble les termes de même nature:

  • variables de même nom
  • puissances de même exposant
  • produits de variables
  • nombres

Exemples:

2洧논 + 5 + 5y + 3 + 3洧논 
⇒ (2洧논+ 3洧논) + 5y + (5 + 3)

5洧논² + 3洧논 + 洧논² + 2洧논
⇒ (5洧논² + 洧논²) + (3洧논 + 2洧논)

a² + 2ab + 2 - 4ab
⇒ a² + (2ab - 4ab) + 2
