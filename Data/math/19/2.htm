Propriétés (1)

- Côtés opposés parallèles 2 à 2
- Côtés opposés de même longueur

- Angles opposés égaux
- Angles consécutifs supplémentaires

 

AB = DC et AD = BC

$$\widehat{A} \text{ = } \widehat{C} \text{ et } \widehat{D} \text{ = } \widehat{B}$$

$$\widehat{A} \text{ + } \widehat{B}\text{ = 180}^\circ$$

$$\widehat{B} \text{ + } \widehat{C}\text{ = 180}^\circ$$

$$\widehat{C} \text{ + } \widehat{D}\text{ = 180}^\circ$$

$$\widehat{D} \text{ + } \widehat{A}\text{ = 180}^\circ$$
